# OpenTelemetry Control Plane

## Purpose

OpenTelemetry Control Plane is an open source platform to manage a fleet of OpenTelemetry Collectors using OPAMP.

## Goals

- Collector registration
- Collector inventory
- Configuration management
- Binary lifecycle management
- Upgrade and rollback
- Agent health monitoring

## Technology

Backend: Go

Frontend: React

Protocol: OPAMP

Deployment: Kubernetes
