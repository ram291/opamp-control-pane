# Glossary

Collector:
Component responsible for receiving, processing and exporting telemetry.

OTLP:
OpenTelemetry Protocol.

OPAMP:
Open Agent Management Protocol.

Receiver:
Accepts telemetry.

Processor:
Transforms telemetry.

Exporter:
Sends telemetry.

Desired State:
State requested by management plane.

Current State:
State running on agent.
